CakeQuest

  • Spring Boot
  • Hibernate/JPA
  • MySQL
  • Thymeleaf
  • REST APIs
  • Web Scraping (Jsoup)
  • PWA Development
  • CI/CD (GitHub Actions, Docker, DigitalOcean)

CakeQuest is a server side rendered progressive web app made with Spring Boot. This project creates a place for home bakers to centralize all of their favorite recipes by simply pasting in the URL. Recipes are then editable, giving bakers the ability to customize a recipe to their own taste. Cake, frosting, and filling recipes can be put together as custom Cake Orders. Each Cake Order automatically generates a unified shopping list, custom tasks, and a suggested timeline, making big cake projects simple and stress-free.

Taniti Website Prototype

  • UI Design (Low & High Fidelity Wireframes)
  • UX Research & Usability Testing
  • Information Architecture & Navigation Design
  • User Feedback Analysis & Iteration

This is a clickable web prototype built using Figma. For this project, I was given a set of requirements and a packet of information about a fictional island called "Taniti." I was tasked with creating a prototype for "Taniti's" website to support the island's growing tourism industry. I created low-fidelity and high-fidelity wireframes, built a clickable prototype, conducted usability testing, analyzed user feedback, iterated on designs, and presented the final high-fidelity wireframes.
